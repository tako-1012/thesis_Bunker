## 追加デバッグ（2回目）

### 問題: 境界値エラー
**発見**: 中規模・大規模テストで全て失敗
**原因**: 最大境界ちょうどの座標が範囲外と判定される
- grid_size=(200, 200, 50)の場合、インデックスは0-199
- 座標(10.0, 10.0, 0.5)はインデックス(200, 200, 5)で範囲外

**解決**: 
- 全テストケースの座標を境界から0.1m内側に調整
- 例: (10.0, 10.0, 0.5) → (9.9, 9.9, 0.5)

### 修正内容（2回目）

#### 1. performance_test.py: テストケース座標修正
```python
# 修正前
("中規模", (100, 100, 30), (0.0, 0.0, 0.5), (5.0, 5.0, 0.5)),
("大規模", (200, 200, 50), (0.0, 0.0, 0.5), (10.0, 10.0, 0.5)),

# 修正後
("中規模", (100, 100, 30), (0.0, 0.0, 0.5), (4.9, 4.9, 0.5)),
("大規模", (200, 200, 50), (0.0, 0.0, 0.5), (9.9, 9.9, 0.5)),
```
