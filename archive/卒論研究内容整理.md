# 🎓 研究室プロジェクト整理

## 📋 プロジェクト概要

**プロジェクト名**: Bunker Robot 研究室プロジェクト
**内容**: 農業用途移動ロボット「Bunker」の自律ナビゲーションシステム
**出典**: 研究室GitHubリポジトリ（そのまま落とし込み）
**研究分野**: ロボティクス、SLAM、農業応用技術

## 🤖 研究対象システム

### Bunker移動ロボット
- **ハードウェア**: AgileX Bunker移動ロボット
- **センサー構成**:
  - Azure Kinect DK（RGB-Dカメラ）
  - Intel RealSense D455（RGB-Dカメラ）
  - Velodyne VLP-16（3D LiDAR）
  - Xsens IMU（慣性計測装置）
  - EMLID GNSS（GPS受信機）

### ソフトウェア構成
- **OS**: Ubuntu 22.04 LTS
- **ROS**: ROS 2 Humble
- **SLAM**: RTAB-Map
- **ナビゲーション**: Nav2
- **シミュレーション**: Unity 2022.3 LTS

---

## 🔬 システム機能

### 1. マルチセンサーSLAMシステム

#### RTAB-Mapによる3D SLAM
- **RGB-D SLAM**: Azure Kinect/RealSenseを使用
- **LiDAR SLAM**: Velodyne VLP-16による点群処理
- **センサーフュージョン**: EKFによる複数センサー統合

#### 実装されたSLAM機能
```bash
# Kinect RTAB-Map
ros2 launch bunker_nav kinect_rtabmap.launch.py

# LiDAR RTAB-Map  
ros2 launch bunker_nav lidar_rtabmap.launch.py

# セマンティックRTAB-Map（トマト認識）
ros2 launch bunker_nav semantic_rtabmap.launch.py
```

### 2. 農業応用機能

#### トマト認識システム
- **セマンティックセグメンテーション**: MMSegmentation使用
- **データセット**: RUGD（Rural-Urban General Dataset）
- **モデル**: OCRNet（Object-Contextual Representations）
- **認識精度**: 農業環境特化モデル

#### 実装された農業機能
```bash
# トマト認識SLAM
ros2 launch bunker_nav semantic.launch.py

# 人間追従機能
ros2 launch bunker_nav follow_human.launch.py

# GPSウェイポイント巡回
ros2 launch bunker_nav demo_gps_waypoint_follower.launch.py
```

### 3. Unity-ROS2連携シミュレーション

#### 高品質3Dシミュレーション
- **Unity 2022.3 LTS**: リアルタイムレンダリング
- **ROS-TCP-Connector**: ROS2-Unity通信
- **URDF-Importer**: ロボットモデルインポート
- **UnitySensors**: センサーシミュレーション

#### 自動ワールド生成機能
```csharp
// TestWorldManagerでプリセット選択
- NavigationTest: ナビゲーションテスト
- ObstacleAvoidance: 障害物回避テスト  
- RaceTrack: レースコース
- OpenField: オープンフィールド
- Urban: 都市環境
```

### 4. ナビゲーションシステム

#### Nav2による自律移動
- **グローバルプランニング**: A*アルゴリズム
- **ローカルプランニング**: DWA（Dynamic Window Approach）
- **障害物回避**: リアルタイム障害物検出
- **ウェイポイントナビゲーション**: GPS座標による自動巡回

## 🚀 使用方法

### 実機でのSLAM実行
```bash
# 1. センサー起動
ros2 launch bunker_bringup azure.launch.py

# 2. SLAM開始
ros2 launch bunker_nav kinect_rtabmap.launch.py

# 3. ナビゲーション
ros2 launch bunker_nav navigation_launch.py
```

### Unityシミュレーション
```bash
# 5つのターミナルで順次実行
ros2 launch bunker_unity launcher.launch.py
ros2 launch bunker_unity spawn_bunker.launch.py
ros2 launch bunker_unity bunker_description.launch.py
ros2 run ros_tcp_endpoint default_server_endpoint
ros2 launch bunker_unity controllers.launch.py
```

### トマト認識SLAM
```bash
ros2 launch bunker_nav semantic_rtabmap.launch.py
```
