# 🤖 AI研究アシスタント用スクリプト

## 📋 研究プロジェクト概要

**プロジェクト名**: Bunker Robot 新機能開発研究
**研究期間**: 卒論研究期間
**目的**: 既存Bunker Robotシステムに新機能を追加し、就活でアピールできる技術的成果を創出

---

## 🎯 研究の位置づけ

### 基盤システム
- **Bunker Robot**: 農業用途移動ロボット（研究室の既存システム）
- **技術スタック**: ROS2 Humble + RTAB-Map + Nav2 + Unity
- **センサー**: Azure Kinect、RealSense、Velodyne VLP-16、IMU、GPS

### 研究方針
- 既存システムを**基盤として活用**
- **新機能を追加**して差別化
- **実機での動作確認**まで含む実践的研究
- **就活でアピール**できる技術的成果の創出

---
