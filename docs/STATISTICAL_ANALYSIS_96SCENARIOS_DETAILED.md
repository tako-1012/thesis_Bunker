## 🎯 Cohen's d効果量の解釈

### 効果量の定義
```
Cohen's d:
  |d| < 0.2    : 無視できる効果 (trivial)
  0.2 ≤ |d| < 0.5 : 小さい効果 (small)
  0.5 ≤ |d| < 0.8 : 中程度の効果 (medium)
  |d| ≥ 0.8    : 大きい効果 (large)
```

### 本研究の結果
- **TA* vs AHA***: d = 0.938 → **非常に大きい効果**
- **TA* vs Theta***: d = 0.925 → **非常に大きい効果**

**意味**: この違いは統計的に有意というだけでなく、**実践的にも極めて重要**な差です。

## ⚠️ 重要な統計的警告

### 1. 非正規分布
TA*とTheta*のデータは**正規分布に従わない**可能性があります。
- **理由**: 計算時間は右に歪んだ分布（シナリオ難度依存）
- **対策**: Welch t検定は正規性に比較的ロバスト
- **推奨**: 非パラメトリック検定（Mann-Whitney U検定）での検証

### 2. 異分散性
3つの手法の分散が大きく異なります：
- AHA*: 分散 = 0.0000838
- Theta*: 分散 = 0.1907
- TA*: 分散 = 541.87

**対策済み**: Welch t検定は異分散を想定した手法なので問題なし

### 3. 外れ値の存在
TA*に以下の外れ値があります：
- 最大値: 105.24秒 (中央値の15倍)
- これはシナリオの複雑性を示唆
