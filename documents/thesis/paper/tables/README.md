## 表作成のベストプラクティス

### 1. データの整理
- 数値の有効桁数を統一
- 単位を明確に記載
- 統計情報（平均±標準偏差）を併記

### 2. 視覚的表現
- 重要な値を**太字**で強調
- 色分けで比較しやすくする
- 適切な罫線を使用

### 3. 情報の完全性
- 必要な情報をすべて含める
- 欠損データは明記
- 実験条件を明確に記載

### 4. 読みやすさ
- 適切な列幅
- 明確なヘッダー
- 論理的な順序
