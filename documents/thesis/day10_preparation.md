## CostCalculatorクラス

### 主要機能
1. **統合コスト計算**: 距離、傾斜、障害物、リスク、平滑化の総合評価
2. **傾斜コスト**: 0-15度（0）、15-25度（線形）、25-35度（急増）、35度以上（不可）
3. **障害物コスト**: あり（1000）、なし（0）
4. **転倒リスクコスト**: 0-1 → 0-10の変換
5. **平滑化コスト**: 急旋回ペナルティ

### 重み付けパラメータ
- **distance**: 1.0（基準）
- **slope**: 3.0（重要度: 高）
- **obstacle**: 5.0（重要度: 最高）
- **risk**: 4.0（重要度: 高）
- **smoothness**: 1.0（重要度: 中）

### コスト計算式
```python
total_cost = (
    base_distance * weight_distance +
    slope_cost * weight_slope +
    obstacle_cost * weight_obstacle +
    risk_cost * weight_risk +
    smoothness_cost * weight_smoothness
)
```
