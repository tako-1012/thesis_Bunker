### Option 2: Hierarchical A* - 中優先度

#### 手法
```
単一精度の探索 → 2段階探索
1. 粗いグリッド（0.5m）で大局的経路
2. 細かいグリッド（0.1m）で局所精細化
```

#### 期待される効果
```
大規模マップでの処理時間: 1/100-1/1000
探索ノード: 1/1000-1/10000
実装難易度: ★★★★☆（高）
```

#### 実装内容
```python
class HierarchicalTAStar(TA-A* Optimized):
    """階層的A*"""
    
    def __init__(self):
        self.coarse_grid = 0.5  # 粗いグリッド
        self.fine_grid = 0.1    # 細かいグリッド
    
    def plan_path(self, start, goal):
        # Step 1: 粗いグリッドで大局的経路
        coarse_path = self._plan_coarse(start, goal)
        
        # Step 2: 各セグメントを細かく精細化
        fine_path = []
        for i in range(len(coarse_path) - 1):
            segment = self._plan_fine(
                coarse_path[i], 
                coarse_path[i+1]
            )
            fine_path.extend(segment)
        
        return fine_path
```

### Option 3: Anytime A* - 低優先度

#### 手法
```
厳密な最適解を求めず、時間に応じて妥協的な解を提供
```

#### 期待される効果
```
時間制約がある場合: 即座に解を提供
処理時間: 柔軟に調整可能
実装難易度: ★★☆☆☆（低）
```

#### 実装内容
```python
class AnytimeTAStar(TA-A* Optimized):
    """Anytime A*"""
    
    def plan_path_anytime(self, start, goal, time_limit):
        """時間制限付き探索"""
        best_path = None
        best_cost = float('inf')
        
        # 重み付きA*で探索
        for w in [10.0, 5.0, 2.0, 1.0]:
            if time.time() > time_limit:
                return best_path
            
            path = self._weighted_search(start, goal, w)
            if path and path.cost < best_cost:
                best_path = path
                best_cost = path.cost
        
        return best_path
```

#### 実装時間
- 調査: 1日
- 実装: 1-2日
- テスト: 1日
- **合計: 3-4日**
