## 2. 統計分析フレームワーク

### 2.1 実装した統計手法

#### ✅ Welch t-test（ウェルチのt検定）

```python
【特徴】
  - 2群の平均値を比較
  - 不等分散に対応（標準的なt-testより堅牢）
  - 帰無仮説: 2つのアルゴリズムに差がない

【結果判定】
  p < 0.05: 統計的に有意な差あり
  p < 0.001: 極めて有意な差あり（実務的に重要）
```

#### ✅ Cohen's d（効果量）

```python
【解釈】
  d < 0.2:    Small effect (小)
  0.2 ≤ d < 0.5: Small effect (小)
  0.5 ≤ d < 0.8: Medium effect (中)
  d ≥ 0.8:    Large effect (大)

【TA* vs 他手法の結果】
  すべて d > 0.9 → 実務的に大きな差
```

#### ✅ 95% 信頼区間（Confidence Interval）

```python
【計算方法】
  CI = mean ± t_critical × (SE)
  SE = std / sqrt(n)
  t_critical は自由度とα=0.05から算出

【解釈】
  「同じ実験を100回繰り返したら、
   95回はこの区間に真の平均値が入る」
```

#### ✅ 記述統計

```python
【計算項目】
  - 平均値 (mean)
  - 標準偏差 (std)
  - 中央値 (median)
  - 四分位範囲 (IQR)
  - 最小値・最大値
  - 歪度 (skewness)
```

### 2.2 実装コード構造

```python
class StatisticalAnalyzer:
    def compute_descriptive_stats(times):
        """記述統計を計算"""
        return {
            'mean': np.mean(times),
            'std': np.std(times),
            'median': np.median(times),
            'q1': np.percentile(times, 25),
            'q3': np.percentile(times, 75),
            'min': np.min(times),
            'max': np.max(times)
        }
    
    def welch_t_test(group1, group2):
        """Welch t-test を実行"""
        t_stat, p_val = scipy.stats.ttest_ind(
            group1, group2, equal_var=False
        )
        return t_stat, p_val
    
    def cohens_d(group1, group2):
        """Cohen's d を計算"""
        # プールされた標準偏差を使用
        return (mean1 - mean2) / pooled_std
    
    def confidence_interval(times, ci=0.95):
        """95% 信頼区間を計算"""
        se = scipy.stats.sem(times)
        ci_range = se * t_critical
        return mean - ci_range, mean + ci_range
```
